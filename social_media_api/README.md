## Deployment

This Django REST API was deployed using Render.

### Deployment Steps
1. Configured production settings (DEBUG=False, ALLOWED_HOSTS).
2. Added Gunicorn and Whitenoise.
3. Created Procfile.
4. Deployed via Render Web Service.
5. Configured PostgreSQL database.
6. Ran migrations and collectstatic.
